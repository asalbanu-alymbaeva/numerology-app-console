# numerology-app-console
Python program to calculate life path numbers and display numerology descriptions based on birth date.

# Numerology Python

This Python program calculates a person's **Life Path Number** based on their birth date and provides the corresponding numerology description. Users can enter their personal information, including name, age, email, and birth date, and the program outputs a personalized numerology report.

## Features
- Store user information using a `Person` class
- Calculate the Life Path Number from birth date
- Retrieve detailed numerology description from `description.txt`
- Display full user info and numerology report
